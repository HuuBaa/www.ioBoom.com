关于博主

  • 博主是自学的Python，做这个网站用于个人日常学习编程时分享心得、记录知识
  • Github地址：github.com/HuuBaa
  • 可以邮箱联系我：ioboom@ioboom.com

网站基本功能

  • 完整的用户系统，支持邮箱验证、重置密码、修改密码等
  • 除了本身用户系统外，支持QQ、微博、Github的第三方账户登录、绑定
  • 文章评论系统，注册后可以自由进行评论
  • 文章按标签分类
  • 实现了Restful Api

关于网站

  • 本网站使用阿里云的ESC云服务器
  • 申请了SSL证书，支持https
  • 后台用supervisor管理网站进程
  • 部署使用Nginx + gunicorn的组合
  • 网站后端使用了Python的Django框架，具体看下面Django相关的说明
  • 网站前端使用了Bootstrap4进行响应式设计

关于Django

  • Python3.6.1+Django2.0.7
  • 使用django-xadmin做后台管理页面的美化
  • 使用django-allauth做第三方登录并自己添加了QQ第三方登录的支持
  • 使用django-restframework 实现 Restful Api 接口

数据库

  • 使用MySQL数据库进行数据的存储
